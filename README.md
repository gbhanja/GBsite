# Academic Portfolio Website

A modern academic portfolio website inspired by professional researcher pages.

## Features

- Responsive Design
- Dark Mode
- Publications
- Teaching
- CV Download
- Gallery
- Mobile Friendly
- GitHub Pages Ready

## Deploy

1. Create GitHub repository
2. Upload files
3. Settings → Pages
4. Deploy from Main Branch

Your site will be live at:

https://gbhanja.github.io/GBsite
